# 🚀 Tech Vault – Online Electronics Shopping Platform

## 🛒 Overview
**Tech Vault** is a modern, high-end e-commerce website designed for premium gadgets and future technology. It simulates a real-world online electronics store with a clean UI, smart search functionality, and interactive design.

> **Tagline:** *Future Tech, Delivered Today.*

---

## 🌐 Website Working

When users open the website, they are welcomed with a visually striking **hero section** showcasing:

- **New Arrivals 2025**
- A futuristic design layout
- Premium branding experience

### 🔍 Smart Search System
- Users can search using keywords like:
  - Laptop
  - Gaming Mouse
  - Headphones
  - Smartwatch
- A **JavaScript-based filtering system** dynamically displays matching products.
- If no product matches → ❌ *"Item not found"*
- Clearing the search → 🔄 resets to featured products

### 🧭 Navigation Flow
- Limited featured products shown initially (clean UI)
- Categories help users explore easily:
  - Electronics
  - Gaming
  - Accessories
  - Smart Devices
  - Audio
